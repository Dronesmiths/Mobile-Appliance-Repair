# Mobile Appliance Repair Service

High-authority static website for Mobile Appliance Repair Service in the Antelope Valley.

## Technical Info
- **Architecture**: Static HTML5/CSS3
- **Deployment**: AWS S3 + CloudFront
- **Optimization**: All images WebP, optimized for PageSpeed.

## Project Structure
- `/factory`: Website Factory V3 engine and scripts.
- `/images`: Optimized assets.
- `/services`: Individual service landing pages.
- `/locations`: Localized SEO landing pages.
- `/blog`: Authority-building content.
